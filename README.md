# 🎓 Agina Academy | أكاديمية أجينا 🚀

مرحباً بكم في **أكاديمية أجينا**، هذا المستودع هو مساحتي التعليمية الخاصة لتوثيق رحلتي في عالم التكنولوجيا، البرمجة، والأمن السيبراني. هنا أضع خلاصة ما أتعلمه من دورات، كتب، وتجارب عملية.

---

## 📚 الأقسام الرئيسية | Main Departments

### 🤖 قسم الذكاء الاصطناعي (AI)
* دراسة أساسيات الـ Machine Learning.
* التعامل مع النماذج اللغوية الكبيرة (LLMs).
* تطبيقات عملية لاستخدام الذكاء الاصطناعي في الأتمتة.

### 💻 قسم البرمجة وتطوير النظم
* **لغة C:** رحلة إتقان الأساسيات والتعامل مع الذاكرة.
* **Python:** استخدامها في بناء الأدوات الأمنية وسكريبتات الأتمتة.
* **Android Development:** استكشاف أنظمة الأندرويد وتخصيصها.

### 🛡️ الأمن السيبراني (Cybersecurity)
* دراسة أدوات اختبار الاختراق (Nmap, Metasploit).
* تحليل البرمجيات الخبيثة (مثل مشروع Pegasus).
* أمن الشبكات والتعامل مع بروتوكولات التشفير.
